Rules

Use your right and left keys to move the paddle to bounce the ball up and break the blocks. If you miss the ball, your score and blocks will reset.

This is a HTML5 version of Atari Breakout, created solely for learning HTML5 Canvas API. Source Code: cakebatterandsprinkles

This app is not optimized for smaller screens, please try using it from your desktop or ipads!
